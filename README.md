# Book-Management
### Application of book management to store the information of books
## This is the backend project of .net with c# 

### The app has
- #### Sign Up
  - name
  - user name (must be unique )
  - email (valid email)
  - password
    
- #### Login
  - User name
  - Password
- #### Add new book
  - book name
  - auther name
  - price
  - catigory
  - publish date
  - rate
  - cover photo
- #### Edit the book information
- #### Delete the book
- #### Show the details of the book
